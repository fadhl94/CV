data-video / <source> paths in index.html to match your own):

  project-1.mp4   Commercial category

Tips:
- Use MP4 (H.264) for the widest browser support.
- Keep files reasonably small (compress with HandBrake) since GitHub Pages
  has bandwidth/size limits and large videos hurt load speed.
- Each video card also needs a matching poster image in /images
  (video-1-poster.jpg, video-2-poster.jpg, video-3-poster.jpg) so the card
  looks good before playback / while metadata loads.
